Example Package Application

Example package with a simple webui to set up a trunk-port

This is an example package with example for request, create and set value.

More information about the NSO JSON-RPC API is available in the NSO userguide.

View the source code of the request here: script.js

Fill out the values and click on "Create Trunk from values".

If the instance name exists, values (interfaces and vlan) will be updated.

When trunk is created in on your Transaction Handle, click "Go to Cisco NSO Commit Manager" to see the changes in the commit manager.

If you in the Commit Manager press "Commit" the data will be commited to your NSO.
